## Формулы площади
### Круг (circle)
Функция area(a) вычисляет площадь круга.
* Параметры:\
r (float): Радиус круга
* Возвращаемое значение:\
Площадь круга (float), вычисленная по формуле S = pi * r * r
* Пример:
```
def area(5):
    return math.pi * r * r
    Output: 78.5
```
### Прямоугольник (rectangle)
Функция area(a) вычисляет площадь прямоугольника.
* Параметры:\
a (float): Длина прямоугольника (неотрицательное число)\
b (float): Ширина прямоугольника (неотрицательное число)
* Возвращаемое значение:\
Площадь прямоугольника (float), вычисленная по формуле S = a * b
* Пример:
```
def area(5, 3):
    return a * b 
    Output: 15.0
```
### Квадрат (square)
Функция area(a) вычисляет площадь квадрата.
* Параметры:\
a (float): Длина стороны квадрата (неотрицательное число)
* Возвращаемое значение:\
Площадь квадрата (float), вычисленная по формуле S = a * a
* Пример:
```
def area(5):
    return a * a
    Output: 25.0
```
### Треугольник (triangle)
Функция area(a) вычисляет площадь треугольника.
* Параметры:\
a (float): Длина основания треугольника (неотрицательное число)\
h (float): Высота треугольника, проведенная к основанию  (неотрицательное число)
* Возвращаемое значение:\
Площадь треугольника (float), вычисленная по формуле S = (a * h) / 2
* Пример:
```
def area(6, 4):
    return a * h / 2
    Output: 12.0
```


## Формулы периметра
### Круг (circle)
Функция perimeter(r) вычисляет длину окружности.
* Параметры:\
r (float): Радиус круга (неотрицательное число)
* Возвращаемое значение:\
Длина окружности, вычисленная по формуле P = 2 * pi * r
* Пример:
```
def perimeter(5):
    return 2 * math.pi * r
    Output: 31.5
```
### Прямоугольник (rectangle)
Функция perimeter(r) вычисляет периметр прямоугольника.
* Параметры:\
a (float): Длина прямоугольника (неотрицательное число)\
b (float): Ширина прямоугольника (неотрицательное число)
* Возвращаемое значение:\
Периметр прямоугольника, вычисленный по формуле P = 2 * (a + b)
* Пример:
```
def perimeter(5, 3):
    return 2(a + b) 
    Output: 16.0
```
### Квадрат (square)
Функция perimeter(r) вычисляет периметр квадрата.
* Параметры:\
a (float): Длина стороны квадрата (неотрицательное число)
* Возвращаемое значение:\
Периметр квадрата, вычисленный по формуле P = 4 * a
* Пример:
```
def perimeter(5):
    return 4 * a
    Output: 20.0
```
### Треугольник (triangle)
Функция perimeter(r) вычисляет периметр треугольника.
* Параметры:\
a (float): Длина первой стороны треугольника (неотрицательное число)\
b (float): Длина второй стороны треугольника (неотрицательное число)\
c (float): Длина третьей стороны треугольника (неотрицательное число)
* Возвращаемое значение:\
Периметр треугольника, вычисленный по формуле P = a + b + c
* Пример:
```
def perimeter(3, 4, 5):
    return a + b + c 
    Output: 12.0
```
